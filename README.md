# Анкета КЦ — Локальное приложение Битрикс24

Локальное приложение для Битрикс24 On-Premise (v22.x+), встраиваемое в карточку лида через плейсмент `CRM_LEAD_DETAIL_TAB`.

## Описание

Приложение позволяет менеджерам Колл-центра заполнять анкету клиента прямо из карточки лида в CRM Битрикс24. Данные сохраняются в пользовательских полях лида (UF_CRM_KC_*) и дублируются в таймлайн. Предусмотрена запись времени клиента через интеграцию с календарём менеджеров.

Интерфейс приложения разделён на **две панели**:
- **Левая** — анкета клиента (все поля UF_CRM_KC_*).
- **Правая** — расписание менеджеров и дополнительные инструменты (разрабатываются).

## Стек технологий

### Frontend
- **Tailwind CSS 4** — утилитарный CSS-фреймворк (подключается через CDN Play)
- **Flowbite** — компонентная библиотека поверх Tailwind (модальные окна, дропдауны, тоггл, тултипы и т.д.)
- Кастомные CSS-стили **не используются** — все стили только через утилиты Tailwind и компоненты Flowbite

### Backend
- **PHP 8.1** — серверная логика, установка приложения, proxy-слой

### Интеграции
- **Bitrix24 REST API** — работа с лидами, полями, таймлайном, событиями календаря
- **BX24.js** — официальный SDK Битрикс24 для iframe-приложений
- **Плейсмент:** `CRM_LEAD_DETAIL_TAB`

### Протокол
- HTTPS (обязательно для iframe-приложений Битрикс24)

## Установка

1. Разместить файлы на HTTPS-сервере
2. В Битрикс24: Разработчикам → Приложения → Добавить → Локальное приложение
3. Указать URL: `https://your-server.ru/anketa-kc/install.php`
4. Выбрать scope: `crm, calendar, user, im, placement`
5. Скопировать `client_id` и `client_secret` в `config.php`

## Пользовательские поля (UF_CRM_KC_*)

Создаются автоматически при установке. Всего **27 полей** в лиде:

| № | FIELDNAME | Тип | Описание |
|---|-----------|-----|----------|
| 1 | KC_FULLNAME | string | ФИО клиента |
| 2 | KC_DEBT_TOTAL | integer | Общий долг |
| 3 | KC_MONTHLY_PAYMENT | integer | Ежемесячный платёж |
| 4 | KC_INCOME_OFFICIAL | enumeration | Официальный доход: Да / Нет |
| 5 | KC_INCOME_OFFICIAL_AMOUNT | integer | Сумма официального дохода (показывается только при Да) |
| 6 | KC_INCOME_UNOFFICIAL | integer | Неофициальный доход |
| 7 | KC_SALARY_CARD | enumeration | Зарплатная карта |
| ... | ... | ... | ... |
| 27 | KC_BOOKED_EVENT_ID | integer | ID события в календаре |

Полный список — в [ТЗ](anketa-kc/docs/tz.md).
