# Анкета (МКЦ + МП) — Локальное приложение Битрикс24

Локальное приложение для портала **https://crm.yurclick.com**, встраивающееся как вкладка в карточку лида CRM Битрикс24.

---

## Назначение

Виджет предназначен для менеджеров колл-центра (МКЦ) и менеджеров по продажам (МП). Он встраивается в карточку лида через плейсмент `CRM_LEAD_DETAIL_TAB` и решает две задачи:

1. **Заполнение анкеты клиента** — 27 пользовательских полей `UF_CRM_KC_*` в карточке лида, сгруппированных по смысловым блокам.
2. **Запись клиента на встречу с МП** — просмотр расписания всех МП с учётом часовых поясов, выбор свободного слота, запуск бизнес-процесса «Назначить встречу».

---

## Расположение в Битрикс24

Приложение регистрируется в интерфейсе Битрикс24 через плейсмент `CRM_LEAD_DETAIL_TAB` — отдельная вкладка с заголовком **«Анкета КЦ и МП»** появляется в детальной карточке каждого лида CRM.

Установка выполняется через `install.php`, который:
- создаёт 27 пользовательских полей лида (`crm.lead.userfield.add`),
- регистрирует плейсмент (`placement.bind`),
- вызывает `BX24.installFinish()` для завершения установки.

## Файлы проекта

```
anketa-kc/
├── assets/
│   ├── app.js          — инициализация: BX24.init, загрузка лида, пользователя
│   ├── form.js         — рендер анкеты (27 полей), валидация, сохранение в лид + таймлайн
│   ├── calendar.js     — расписание МП, выбор слота, запуск БП «Назначить встречу»
│   ├── mp-config.js    — конфигурация МП (список, часовые пояса, рабочие дни, часы)
│   ├── cities.js       — справочник 922 городов России с UTC-смещениями
│   └── polling.js      — заглушка (автообновление отключено, обновление вручную)
├── config.php          — константы: PORTAL_URL, WEBHOOK_URL, BP_TEMPLATE_ID и др.
├── index.php           — основной iframe-виджет (HTML + Tailwind + подключение JS)
├── install.php         — установщик: создание 27 UF-полей лида + placement.bind
├── uninstall.php       — деинсталлятор: placement.unbind + опциональное удаление полей
└── manifest.json       — манифест приложения (scopes, placements)
```

### Описание файлов

| Файл | Назначение |
|---|---|
| `assets/app.js` | Точка входа. Вызывает `BX24.init()`, получает текущий лид (`crm.lead.get`), текущего пользователя (`user.current`), инициализирует форму и календарь. |
| `assets/form.js` | Рендерит HTML-форму из 27 полей. Обрабатывает валидацию (10 обязательных полей). При сохранении вызывает `crm.lead.update` и добавляет запись в таймлайн через `crm.timeline.comment.add`. |
| `assets/calendar.js` | Загружает расписание всех МП через `calendar.event.get`. Строит таблицу слотов с учётом TZ клиента и МП. При бронировании запускает БП через `bizproc.workflow.start` с параметрами `DateTime`, `DateTimeClient`, `CalendarMenager`, `ConsultationChannel`. |
| `assets/mp-config.js` | Статичная конфигурация менеджеров по продажам: ID пользователей Битрикс24, часовые пояса, рабочие дни недели, рабочие часы. |
| `assets/cities.js` | Справочник 922 городов России с UTC-смещениями. Используется для определения часового пояса клиента по выбранному городу. |
| `assets/polling.js` | Заглушка. Экспортирует `startPolling()` — пустую функцию. Автообновление расписания отключено для экономии API-лимитов. |
| `config.php` | PHP-константы проекта. Передаются в JS через `window.APP_CONFIG` в `index.php`. |
| `index.php` | Основной iframe. Подключает Tailwind CSS v4 CDN, Flowbite 2, BX24 JS SDK и все JS-модули из `assets/`. |
| `install.php` | Создаёт 27 пользовательских полей лида и регистрирует плейсмент `CRM_LEAD_DETAIL_TAB`. Вызывается Битрикс24 при первой установке приложения. |
| `uninstall.php` | Отвязывает плейсмент через `placement.unbind`. Опционально удаляет все поля `UF_CRM_KC_*` через `crm.lead.userfield.delete`. |
| `manifest.json` | Манифест локального приложения: slug, version, name, description, install_url, handler_url, scope, placements. |

## Установка

1. Загрузите файлы приложения на сервер в директорию `/anketa-kc/`.
2. Перейдите в Битрикс24: **Приложения → Разработчикам → Добавить приложение → Локальное приложение**.
3. В поле «Ссылка для установки» укажите:
   ```
   https://crm.yurclick.com/anketa-kc/install.php
   ```
4. Нажмите «Сохранить». Битрикс24 откроет `install.php` в iframe.
5. `install.php` автоматически:
   - создаст 27 пользовательских полей лида через `crm.lead.userfield.add`,
   - зарегистрирует вкладку через `placement.bind` с плейсментом `CRM_LEAD_DETAIL_TAB`,
   - вызовет `BX24.installFinish()`.
6. После установки вкладка **«Анкета КЦ и МП»** появится в каждой карточке лида.

### Настройка config.php

Перед установкой проверьте значения констант в `config.php`:

| Константа | Значение по умолчанию | Описание |
|---|---|---|
| `PORTAL_URL` | `https://crm.yurclick.com` | URL вашего портала |
| `WEBHOOK_URL` | `https://crm.yurclick.com/rest/6/...` | Входящий вебхук (для будущих серверных вызовов) |
| `BP_TEMPLATE_ID` | `40` | ID шаблона БП «Назначить встречу» |
| `SALES_DEPT_ID` | `1` | ID отдела продаж |
| `SLOT_DURATION_MIN` | `60` | Длительность слота (минуты) |
| `SLOT_HORIZON_DAYS` | `7` | Горизонт отображения расписания (дни) |
| `MIN_SLOTS_PER_DAY` | `1` | Минимум слотов для отображения дня |
| `CLIENT_HOUR_MIN` | `9` | Минимальный час по TZ клиента |
| `CLIENT_HOUR_MAX` | `20` | Максимальный час по TZ клиента |

## Удаление

1. Перейдите в Битрикс24: **Приложения → Установленные → Анкета КЦ и МП → Удалить**.
2. Битрикс24 откроет `uninstall.php`, который:
   - отвяжет плейсмент через `placement.unbind`,
   - опционально удалит все поля `UF_CRM_KC_*` через `crm.lead.userfield.delete`.

> **Важно:** удаление полей необратимо. Все данные анкет во всех лидах будут утеряны. По умолчанию удаление полей отключено — раскомментируйте соответствующий блок в `uninstall.php`, если это необходимо.

## Технический стек

| Компонент | Версия / Описание |
|---|---|
| **PHP** | Бэкенд: `config.php`, `install.php`, `uninstall.php`. Генерирует `window.APP_CONFIG` для JS. |
| **JavaScript** | ES2020+, Vanilla JS (без фреймворков). Модульная структура: 6 файлов в `assets/`. |
| **Tailwind CSS** | v4 CDN — утилитарные классы для стилизации интерфейса. |
| **Flowbite** | v2 — компоненты UI на основе Tailwind (модальные окна, таблицы, кнопки). |
| **BX24 JS SDK** | Официальный SDK Битрикс24 для вызовов REST API из iframe-приложений. |
| **REST API Битрикс24** | Вызовы: `crm.lead.get/update`, `crm.lead.userfield.add/delete`, `calendar.event.get`, `bizproc.workflow.start`, `placement.bind/unbind`, `user.current`. |

### Скоупы приложения

| Скоуп | Для чего используется |
|---|---|
| `crm` | Чтение и запись лидов, создание/удаление пользовательских полей |
| `calendar` | Чтение расписания МП для отображения свободных слотов |
| `user` | Получение данных текущего пользователя (ФИО менеджера КЦ) |
| `im` | Зарезервировано для будущих уведомлений через внутренний чат |
| `placement` | Регистрация вкладки `CRM_LEAD_DETAIL_TAB` в карточке лида |
| `bizproc` | Запуск бизнес-процесса «Назначить встречу» при бронировании слота |

### О manifest.json

Файл `manifest.json` содержит следующие ключи:

- `slug` — уникальный идентификатор приложения (`anketa-kc-mp`).
- `version` — семантическая версия (`1.0.0`).
- `name` — локализованное название (ru / en).
- `description` — локализованное описание (ru).
- `install_url` — URL страницы установки (`install.php`), вызывается Битрикс24 при первой установке.
- `handler_url` — URL основного iframe-обработчика (`index.php`).
- `scope` — массив запрашиваемых разрешений: `crm`, `calendar`, `user`, `im`, `placement`, `bizproc`.
- `placements` — массив точек встраивания. Единственный плейсмент: `CRM_LEAD_DETAIL_TAB` с заголовком «Анкета КЦ и МП».

> **Примечание:** формат JSON не поддерживает комментарии, поэтому описание ключей манифеста приведено только в этом README.
