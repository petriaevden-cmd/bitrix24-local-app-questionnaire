# Анкета КЦ — Локальное приложение Битрикс24

Локальное приложение для Битрикс24 On-Premise (v22.x+), встраиваемое в карточку лида через плейсмент `CRM_LEAD_DETAIL_TAB`.

## Описание

Приложение позволяет менеджерам Колл-центра заполнять анкету клиента прямо из карточки лида в CRM Битрикс24. Данные сохраняются в пользовательских полях лида (UF_CRM_KC_*) и дублируются в таймлайн. Предусмотрена запись времени клиента через интеграцию с календарём менеджеров.

## Стек технологий

- **Backend:** PHP 8.1
- **Frontend:** Vanilla JS (ES6), HTML5, CSS3
- **SDK:** BX24.js (Битрикс24 REST API)
- **Протокол:** HTTPS
- **Плейсмент:** `CRM_LEAD_DETAIL_TAB`

## Установка

1. Разместить файлы на HTTPS-сервере
2. В Битрикс24: Разработчикам → Приложения → Добавить → Локальное приложение
3. Указать URL: `https://your-server.ru/anketa-kc/install.php`
4. Выбрать scope: `crm, calendar, user, im, placement`
5. Скопировать `client_id` и `client_secret` в `config.php`

## Пользовательские поля (UF_CRM_KC_*)

Создаются автоматически при установке. Всего 26 полей в лиде:

| № | FIELDNAME | Тип | Описание |
|---|-----------|-----|----------|
| 1 | KC_FULLNAME | string | ФИО клиента |
| 2 | KC_DEBT_TOTAL | integer | Общий долг |
| 3 | KC_MONTHLY_PAYMENT | integer | Ежемесячный платёж |
| ... | ... | ... | ... |
| 26 | KC_BOOKED_EVENT_ID | integer | ID события в календаре |

Полный список — в [ТЗ](docs/tz.md).
