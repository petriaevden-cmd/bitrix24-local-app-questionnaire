# Техническое задание — Анкета КЦ

## Проект

Локальное приложение Bitrix24 для анкеты колл-центра. Открывается как вкладка (таб) в карточке лида CRM. Интерфейс — iframe поверх Bitrix24 REST API и BX24 JS SDK. Backend — PHP.

## Утверждённый стек

| Направление | Решение |
|---|---|
| Frontend styles | Tailwind CSS 4 (CDN) |
| UI components | Flowbite 2 |
| CRM / portal | Bitrix24 REST API + BX24 JS SDK |
| Backend | PHP |

## Требования к frontend

### Общие правила
- Любой новый экран или переработка строится на **Tailwind CSS 4**.
- Стандартные UI-элементы берутся из **Flowbite** (dropdown, tabs, buttons, forms, modals, alerts, cards).
- Самописные CSS-файлы (`tokens.css`, `style.css`) **не подключаются** в `index.php`.
- Кастомные стили допустимы только точечно для Bitrix24 iframe/embed-особенностей.
- Новый дизайн **не проектируется с нуля** при каждой итерации — UI развивается внутри одного стека.

### Layout

Основной экран — двухколоночный:
- **Левая колонка (55%)** — форма анкеты по блокам.
- **Правая колонка (45%)** — расписание менеджеров, будущие инструменты.

Правая панель расширяемая: скоринг, документы, история изменений, подсказки для менеджера.

### Блоки формы (левая колонка)

| # | Блок | Поля |
|---|---|---|
| 1 | Финансовые данные | ФИО, сумма долга, платёж, доходы, зарплатная карта |
| 2 | Кредитная история | Кредиторы, залог, просрочки, ФССП, имущество, сделки |
| 3 | Личные данные | Работа, семья, дети, совм. имущество, судимости, ООО, ИП |
| 4 | Заметки менеджера | Исключение КМ, боль, возражения, доп. комментарий |
| 5 | Запись к специалисту | Менеджер, время, ID события |

### Компоненты Flowbite для использования

- `Tabs` — для переключения правой панели (расписание / фишки)
- `Card` — для блоков формы
- `Button` — кнопки действий
- `Input`, `Select`, `Textarea` — поля формы
- `Alert` — сообщения об ошибках и успехе
- `Spinner` — состояние загрузки
- `Badge` — статусы слотов

## Требования к интеграции

- Получение данных лида — `crm.lead.get` через BX24 JS SDK.
- Сохранение анкеты — `crm.lead.update` (UF_CRM_KC_* поля).
- Таймлайн — `crm.timeline.comment.add` после сохранения.
- Расписание — `calendar.event.get` / `calendar.event.add`.
- Уведомление менеджера — `im.notify`.
- Backend — PHP (config, install, proxy, uninstall).

## Совместимость

Tailwind CSS v4 требует Safari 16.4+, Chrome 111+, Firefox 128+.
